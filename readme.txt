Easy to use plugin for adding a widget for all your referral programs.
 
== Description ==
 
With this plugin you can add any of your referral links to promote on your site. You can add any of the referral program of the cryptocurrencies exchange or any other links that you consider.

The plugin has a javascript library to randomize the deliver of any creative that you upload, even if you are using a CDN or some other cache system, the plugin still works and will still randomdly serve any referral link that you upload.
